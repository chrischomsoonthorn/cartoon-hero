Example 1 - string text delivered through 'span' element with adjusted formatting from class labelling.

This is an example of test text with class name 'test1' under the Span element. as you can see, as soon as I change the span's class to the new class, which is now 'test2', the font and colours change . This can look seemless as if this text was all part of one string but actually it is several spans concatenated together and assigned different classes.
